flex: 1 1 420px; এর মানে — shorthand ভাঙা:

flex-grow: 1 → available space থাকলে item বাড়তে পারে
flex-shrink: 1 → স্পেস কম হলে item ছোট হতে পারে
flex-basis: 420px → শুরুতে কত জায়গা নেবে (preferred size)
সংক্ষেপে: element শুরুতে 420px নেবে, কিন্তু container-এ জায়গা থাকলে বড় হবে এবং জায়গা কম হলে ছোট হবে, proportionalভাবে অন্যান্য flex items-এর সাথে ভাগ করে নেবে.
